### `main.scss` 색상 변수 설명

`main.scss` 파일에 정의된 색상 변수들은 블로그의 라이트 모드와 다크 모드 스타일을 결정합니다. 각 변수가 어떤 요소의 색상을 담당하는지 자세히 설명해 드리겠습니다.

---

### 1. 라이트 모드 색상 (`Light Colors`)

라이트 모드(기본 테마)에서 사용되는 색상 변수들입니다.

| 변수명 | 설명 |
| --- | --- |
| `$text-base-color` | 블로그의 기본 텍스트 색상입니다. |
| `$text-link-blue` | 일반적인 링크 텍스트 색상입니다. |
| `$text-link-blue-active` | 링크에 마우스를 올리거나 클릭했을 때의 색상입니다. |
| `$bloquote-border` | 인용문(blockquote)의 왼쪽에 표시되는 테두리 색상입니다. |
| `$blockquote-text` | 인용문 내부의 텍스트 색상입니다. |
| `$inline-code-text` | `<code>` 태그와 같이 한 줄로 된 코드 블록의 텍스트 색상입니다. |
| `$inline-code-bg` | 한 줄 코드 블록의 배경색입니다. |
| `$toc-bg` | 목차(Table of Contents)의 배경색입니다. |
| `$go-top-bg` | '맨 위로 가기(Go to top)' 버튼의 배경색입니다. |
| `$go-top-bg-hover` | '맨 위로 가기' 버튼에 마우스를 올렸을 때의 배경색입니다. |
| `$black` | 제목 등 강조되는 텍스트에 사용되는 검은색 계열 색상입니다. |
| `$light` | 테이블 테두리 등 밝은 회색 계열의 테두리 색상입니다. |
| `$smoke` | 링크의 밑줄 등 연한 회색 계열 색상입니다. |
| `$gray` | 메타 정보(날짜, 작성자 등)에 사용되는 회색 텍스트 색상입니다. |
| `$white` | 블로그의 전체 배경색 등 흰색입니다. |

---

### 2. 다크 모드 색상 (`Dark Colors`)

다크 모드에서 사용되는 색상 변수들입니다.

| 변수명 | 설명 |
| --- | --- |
| `$dark-text-base-color` | 다크 모드에서의 기본 텍스트 색상입니다. |
| `$dark-text-link-blue` | 다크 모드에서의 링크 텍스트 색상입니다. |
| `$dark-text-link-blue-active`| 다크 모드에서 링크에 마우스를 올리거나 클릭했을 때의 색상입니다. |
| `$dark-bloquote-border` | 다크 모드에서 인용문의 테두리 색상입니다. |
| `$dark-blockquote-text` | 다크 모드에서 인용문 내부의 텍스트 색상입니다. |
| `$dark-inline-code-text` | 다크 모드에서 한 줄 코드 블록의 텍스트 색상입니다. |
| `$dark-inline-code-bg` | 다크 모드에서 한 줄 코드 블록의 배경색입니다. |
| `$dark-toc-bg` | 다크 모드에서 목차의 배경색입니다. |
| `$dark-go-top-bg` | 다크 모드에서 '맨 위로 가기' 버튼의 배경색입니다. |
| `$dark-go-top-bg-hover` | 다크 모드에서 '맨 위로 가기' 버튼에 마우스를 올렸을 때의 배경색입니다. |
| `$dark-black` | 다크 모드의 전체 배경색입니다. (현재 남색으로 설정된 부분) |
| `$dark-white` | 다크 모드에서 제목 등 강조되는 텍스트 색상입니다. |
| `$dark-light` | 다크 모드에서 테이블 테두리 등 밝은 테두리 색상입니다. |
| `$dark-smoke` | 다크 모드에서 링크 밑줄 등 연한 회색 계열 색상입니다. |
| `$dark-gray` | 다크 모드에서 메타 정보(날짜, 작성자 등)에 사용되는 회색 텍스트 색상입니다. |

---

이 변수들의 값을 수정하면 블로그의 전체적인 색상 테마를 원하시는 대로 변경할 수 있습니다.
